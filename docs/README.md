# Docs

本目录用于放置 AutoDrive Insight Agent 的补充文档。

当前 README 已覆盖项目展示所需的主要内容，包括项目背景、技术栈、系统架构、Agent 工作流、数据字段、规则引擎、严重度分级、AI 分析、API、测试方式和简历描述。

后续可按需补充：

- `architecture.md`：更详细的系统架构与模块边界
- `rules.md`：规则引擎规则表、阈值来源和扩展方式
- `severity-model.md`：严重度评分因子、等级映射和示例
- `api.md`：API 请求与响应示例

> 说明：项目数据为模拟数据，不包含真实车企、真实车辆或真实用户数据。
