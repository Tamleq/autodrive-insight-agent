# AutoDrive Insight Agent Frontend

React + TypeScript + Vite frontend for the AutoDrive Insight Agent demo.

## Run Locally

```bash
npm install
npm run dev
```

The Vite dev server runs at `http://localhost:5173`.

Start the backend separately on `http://localhost:8000`; frontend API calls to `/api/*` are proxied there by `vite.config.ts`.

## Build

```bash
npm run build
```
