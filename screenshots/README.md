# Screenshots

本目录用于存放 GitHub README 展示截图。

建议截图命名：

| 文件名 | 页面 |
| --- | --- |
| `dashboard.png` | Dashboard，展示总事件数、风险分布、问题类型 Top 5 和高风险事件列表 |
| `event-list.png` | 事件列表，展示场景、接管、AEB 筛选 |
| `event-detail.png` | 事件详情，展示关键指标、规则命中、严重度分级和 AI 分析 |
| `report.png` | Markdown 报告页，展示复制与下载报告能力 |

在截图补齐后，可将 README 中“页面截图占位”章节替换为实际图片引用，例如：

```markdown
![Dashboard](screenshots/dashboard.png)
```
